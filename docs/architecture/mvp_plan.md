# MVP : CaisseCash - Consulting Edition (IEL)

## 📌 Vision Stratégique
Le logiciel **CaisseCash** est le tableau de bord officiel de l'**ISS Economics Lab (IEL)** pour le suivi de la santé financière des AGR.
- **Entrée** : Données auditées importées en CSV ou saisies nativement.
- **Sortie** : Application mobile et desktop intuitive pour une lecture immédiate de la performance.

## ✅ Fonctionnalités Livrées (MVP Terminé)

### 1. Import & Structure des Données
- **Import CSV Intelligent** : Chargement des données avec détection auto du format et mapping des portefeuilles.
- **Conformité au Schéma** : Respect strict du schéma de données IEL standardisé.
- **Persistance SQLite** : Stockage local performant et fiable.

### 2. Vue "Tantara" (Historique Mensuel)
- **Sélecteur de Mois Horizontal** : Navigation fluide par défilement.
- **Groupement Quotidien** : Affichage clair des transactions par jour avec sous-totaux.
- **Barre de Résumé** : Visualisation instantanée des flux mensuels.

### 3. Vue "Témoin" (Analyse de Performance)
- **Tableau de Bord** : Graphiques d'évolution du solde et distribution des dépenses.
- **Indicateur de Santé** : Lecture rapide de la situation financière globale.

### 4. Recherche & Filtrage
- **Moteur de recherche** : Filtrage par texte ou par date précise.

## 🚀 Prochaines Étapes (Post-MVP)
- **Saisie Intelligente (Smart Entry)** : Autocomplétion et aide à la saisie pour réduire les erreurs.
- **Génération d'APK Android** : Finalisation du pipeline de build pour la distribution mobile.
- **Export de Backup** : Sauvegarde externe des données SQLite.
