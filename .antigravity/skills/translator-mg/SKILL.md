# 🇲🇬 Instructions : Translator (Malagasy)

Ce skill permet à l'agent de traduire et de structurer les chaînes de caractères pour l'application Cashew.

## 📋 Directives de Traduction
1. **Ton** : Professionnel mais simple, adapté aux commerçants de proximité (AGR).
2. **Terminologie** :
   - `Dépense` -> `Vola nivoaka`
   - `Recette / Entrée` -> `Vola niditra`
   - `Historique` -> `Tantara`
   - `Caisse / Solde` -> `Tahiry / Vola eo am-pelatanana`
   - `Unité` -> `Sora-panakalozana`
3. **Clarté** : Éviter le langage trop technique ou archaïque. Privilégier le langage courant mais respectueux.

## 🛠️ Actions supportées
- **Traduction de clés de dictionnaire** : Traduire un fichier `translations.py` vers le malagasy.
- **Révision de labels** : Vérifier que les noms d'items ou de catégories sont compréhensibles par un utilisateur local.
