# IEL - Expense Tracker (CaisseCash)

Application Flutter de **suivi de dépenses** multi-plateforme. Gère les entrées/sorties financières, le suivi de caisse et la catégorisation des transactions.

## Fonctionnalités

- Suivi des dépenses et revenus
- Catégorisation des transactions
- Base de données locale SQLite
- Export de données
- Interface desktop Linux, Web et Android

## Stack technique

- **Framework** : Flutter (Dart)
- **Architecture** : Riverpod (Provider)
- **Base de données** : SQLite (sqflite)
- **UI** : Material 3, Google Fonts (Outfit)
