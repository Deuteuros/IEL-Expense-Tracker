* Rôle de l'Orchestrateur
Ce fichier est le point d'entrée de l'application. Il gère la navigation entre les vues, le thème global, et les interactions transversales comme l'ajout de transactions via un dialogue flottant.

* Structure de l'Application
L'application est structurée autour d'une =NavigationBar= et d'un conteneur central qui change dynamiquement de contenu.

#+begin_src python :tangle ../main.py
import flet as ft
import database
from views.summary import get_summary_view
from views.history import get_history_view
from views.charts import get_charts_view

def main(page: ft.Page):
    # Setup global
    page.title = "Cashew Expense Tracker"
    page.theme_mode = ft.ThemeMode.LIGHT
    page.theme = ft.Theme(color_scheme_seed="green", use_material3=True)
    
    database.init_db()
#+end_src

* Logique de Navigation
Le changement d'onglet déclenche la fonction =navigate=, qui recalcule la vue demandée et met à jour la page.

#+begin_src python :tangle ../main.py
    def navigate(e):
        index = e.control.selected_index
        if index == 0:
            main_content.content = get_summary_view()
        elif index == 1:
            main_content.content = get_history_view()
        elif index == 2:
            main_content.content = get_charts_view()
        page.update()
#+end_src
